Integritetspolicy för KoaLens

Senast uppdaterad: 18 april 2025

Denna integritetspolicy beskriver hur KoaLens ("vi", "oss" eller "vår") samlar in, använder och skyddar information när du använder vår mobilapplikation KoaLens ("Appen").

Information vi samlar in

Vi strävar efter att minimera insamlingen av personuppgifter. Appen samlar in följande information:

  • Videoinspelningar av ingredienslistor: När du använder videoanalysfunktionen skickas videoinspelningen till Google Gemini API för bearbetning. Videoinspelningen behandlas temporärt på våra servrar innan den skickas vidare till Google Gemini API för att identifiera ingredienser och avgöra vegansk status. Vi lagrar inte själva videoinspelningarna permanent efter analysen.
  • Analysresultat och historik: Resultaten från dina skanningar (inklusive identifierade ingredienser, status och tidpunkt) sparas lokalt på din enhet för att du ska kunna komma åt din historik.
  • Användardefinierade bevakade ingredienser: Ingredienser du lägger till för bevakning sparas lokalt på din enhet.
  • Användarkontoinformation (om tillämpligt): För att möjliggöra funktioner som användningsräknare eller potentiella framtida funktioner som synkronisering eller premiumåtkomst, kan vi samla in och lagra en anonymiserad användaridentifikator (User ID) kopplad till din enhet eller ditt konto via vår backend-leverantör Supabase. Om du registrerar ett konto kan även din e-postadress samlas in.
  • Anonym användningsdata: Vi kan samla in anonym, aggregerad data om hur Appen används (t.ex. vilka funktioner som är populära, kraschrapporter) via verktyg som Firebase Analytics för att förbättra Appens prestanda och användarupplevelse. Denna data kan inte kopplas till dig som individ.

Hur vi använder informationen

  • För att tillhandahålla Appens kärnfunktioner (videoanalys, E-nummerkoll, historik, bevakning, användningsräknare).
  • För att analysera och förbättra Appens funktion och användarvänlighet (via anonym data).
  • För att kommunicera med Googles servrar (Gemini API) för att utföra AI-analysen.
  • För att kommunicera med våra egna backend-servrar (hanterade via Supabase) för att hantera användningsräknare och potentiella kontofunktioner.

Datalagring och säkerhet

Din skanningshistorik och bevakade ingredienser lagras endast lokalt på din enhet. Användar-ID och relaterad data för användningsräknare lagras säkert hos vår backend-leverantör Supabase. Vi vidtar rimliga tekniska och organisatoriska åtgärder för att skydda den information som behandlas via oss och våra tredjepartstjänster, men vi kan inte garantera absolut säkerhet, särskilt vid dataöverföring över internet.

Tredjepartstjänster

Appen använder följande tredjepartstjänster:

  • Google Gemini API: För ingrediensanalys via AI. Din användning omfattas av Googles integritetspolicy och användarvillkor. Data kan överföras till Googles servrar globalt.
  • Supabase: För backend-funktioner som hantering av användningsräknare och användaridentifikation. Deras databehandling regleras av Supabases integritetspolicy.
  • Google Play: För distribution av Appen. Användning regleras av Google Plays villkor.
  • Firebase Analytics (eller liknande): För insamling av anonym användningsdata. Regleras av Googles policyer.

Dina rättigheter

Eftersom den mesta datan (historik, bevakningar) lagras lokalt har du full kontroll över den. Du kan när som helst rensa din skanningshistorik och ta bort bevakade ingredienser direkt i Appen. Om du avinstallerar Appen raderas all lokal data.

Om du har skapat ett konto eller om data kopplad till ditt anonymiserade User ID finns lagrad på våra servrar (via Supabase), har du rätt att begära tillgång till eller radering av denna data. Kontakta oss på koalens.app@gmail.com för att göra en sådan begäran.

Barns integritet

Vår App riktar sig inte till personer under 13 år (eller motsvarande minimiålder i relevant jurisdiktion). Vi samlar inte medvetet in personligt identifierbar information från barn under 13 år. Om du är förälder eller vårdnadshavare och blir medveten om att ditt barn har försett oss med personuppgifter utan ditt samtycke, vänligen kontakta oss. Om vi blir medvetna om att vi har samlat in personuppgifter från barn utan verifierbart föräldramedgivande, vidtar vi åtgärder för att ta bort den informationen från våra servrar och system.

Ändringar i denna policy

Vi kan uppdatera denna integritetspolicy från tid till annan. Vi meddelar dig om väsentliga ändringar genom att publicera den nya policyn i Appen eller på vår webbplats.

Kontakt

Om du har frågor om denna integritetspolicy, vänligen kontakta oss på koalens.app@gmail.com.
