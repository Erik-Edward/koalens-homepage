Integritetspolicy för KoaLens

Senast uppdaterad: torsdag, 17 april 2025

Denna integritetspolicy beskriver hur KoaLens ("vi", "oss" eller "vår") samlar in, använder och skyddar information när du använder vår mobilapplikation KoaLens ("Appen").

Information vi samlar in

Vi strävar efter att minimera insamlingen av personuppgifter. Appen samlar in följande information:

  • Videoinspelningar av ingredienslistor: När du använder videoanalysfunktionen skickas videoinspelningen till Google Gemini API för bearbetning för att identifiera ingredienser och avgöra vegansk status. Vi lagrar inte själva videoinspelningarna permanent.
  • Analysresultat och historik: Resultaten från dina skanningar (inklusive identifierade ingredienser, status och tidpunkt) sparas lokalt på din enhet för att du ska kunna komma åt din historik. Denna data lämnar inte din enhet om du inte aktivt väljer att dela den.
  • Användardefinierade bevakade ingredienser: Ingredienser du lägger till för bevakning sparas lokalt på din enhet.
  • Anonym användningsdata: Vi kan samla in anonym, aggregerad data om hur Appen används (t.ex. vilka funktioner som är populära, kraschrapporter) för att förbättra Appens prestanda och användarupplevelse. Denna data kan inte kopplas till dig som individ.

Hur vi använder informationen

  • För att tillhandahålla Appens kärnfunktioner (videoanalys, E-nummerkoll, historik, bevakning).
  • För att analysera och förbättra Appens funktion och användarvänlighet (via anonym data).
  • För att kommunicera med Googles servrar (Gemini API) för att utföra AI-analysen.

Datalagring och säkerhet

Din skanningshistorik och bevakade ingredienser lagras endast lokalt på din enhet. Vi vidtar rimliga åtgärder för att skydda den information som behandlas via tredjepartstjänster som Google Gemini, men vi kan inte garantera absolut säkerhet.

Tredjepartstjänster

Appen använder Google Gemini API för ingrediensanalys. Din användning av denna funktion omfattas även av Googles integritetspolicy och användarvillkor. Appen distribueras via Google Play, vars användning regleras av deras villkor.

Dina rättigheter

Eftersom den mesta datan lagras lokalt har du full kontroll över den. Du kan när som helst rensa din skanningshistorik och ta bort bevakade ingredienser direkt i Appen. Om du avinstallerar Appen raderas all lokal data.

Ändringar i denna policy

Vi kan uppdatera denna integritetspolicy från tid till annan. Vi meddelar dig om väsentliga ändringar genom att publicera den nya policyn i Appen eller på vår webbplats.

Kontakt

Om du har frågor om denna integritetspolicy, vänligen kontakta oss på kontakt@koalens.se.
